页面说明
行政总览页是 AOP 平台的核心入口，按模块聚合呈现全部行政运营事项，支持列表/看板双视图切换。
模块选择器
Pill 组件按模块组分组，选中态使用 accent-light 底色 + accent 色边框。每组首个 pill 默认激活。点击切换数据区域内容。
视图切换
列表视图以 Table 呈现完整字段；看板视图按状态分为四列，卡片显示摘要信息。切换时无刷新，使用 CSS 动画过渡。
