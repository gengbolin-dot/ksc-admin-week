# AOP平台 V4 — 行政运营管理平台

> Admin Operations Platform · 完整可交互原型

## 📋 项目概述

AOP（行政运营平台）是一个统一的行政工作管理平台，核心数据模型为 **Work Item（工作事项）**，所有页面均为该单一数据模型的不同投影视图。

### V4 更新内容

- 🏗️ **侧边栏深色主题** — 深蓝黑背景(#0F172A)、浅灰文字
- 📝 **完整文字编辑器** — 段落样式/字号/文字颜色/高亮背景/对齐/列表
- 🖼️ **图片自适应布局** — 根据数量自动缩放、支持拖拽调整大小
- 🔗 **重点专项外部链接** — 卡片支持配置外部URL跳转
- 💾 **数据持久化** — 编辑内容/链接/封面图自动保存到localStorage
- 🤖 **AI绩效模块** — 展示行政部AI组织绩效目标
- 💰 **年度预算升级** — Excel导入、详细数据表格、Q1-Q4完整展示
- 📊 **经营复盘** — 季度回顾、演示模式

### 设计原则

- **Single Source of Truth** — 统一数据源，消除多系统信息孤岛
- **Minimal Input / Max Output** — 最小输入，最大输出
- **Push Over Pull** — 推送优于拉取
- **Type-Driven Templates** — 类型驱动模板
- **Role-Based Views** — 基于角色的视图
- **Linear-Style Aesthetics** — Linear 风格极简美学

## 🎨 技术特点

- **纯 HTML + CSS 原型** — 无第三方依赖（仅 SheetJS 用于 Excel 导入）
- **CSS 变量系统** — 统一色彩/间距/圆角设计令牌
- **Linear 风格 UI** — 极简、留白、精致的信息密度控制
- **内嵌数据集** — JS 数据驱动渲染，演示真实数据效果
- **完整交互** — 导航切换、文字编辑、图片上传/拖拽、灯箱查看、PPT演示
- **数据持久化** — localStorage 保存编辑内容、链接、封面图

## 📊 主要功能模块

### 我的工作 (Panel 02)
- 面包屑导航 + 操作按钮
- 周报周期卡片（W28、进度条、一键提交）
- Tab 切换（本周工作/历史周报）
- 分类筛选（全部/重点专项/日常运营）
- 任务卡片（状态徽章、进度条、截止日期）
- 附件上传（图片/视频/PDF）

### 行政总览 (Panel 04)
- 模块筛选 tabs（职场运营/文化运营/数字化运营）
- 列表/看板视图切换
- 事项表格（名称/类型/负责人/状态/完成度）

### 重点专项 (Panel 05)
- 项目卡片（封面图、指标、描述）
- 点击查看/编辑（飞书风格在线编辑器）
- 外部链接跳转
- 数据持久化（链接/指标/封面图）

### 年度预算 (Panel 13)
- 概览视图（分类汇总、进度条、月度趋势图）
- 详细数据表格（Q1-Q4季度、1-12月明细）
- Excel/CSV 导入功能
- 年份切换

### 行政经营复盘 (Panel 12)
- Q2季度回顾（8个板块）
- 图片轮播、灯箱查看
- PPT演示模式
